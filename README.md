# GNU FM

A music community platform in PHP. Powers Libre.fm (2009-) -- see https://github.com/libre-fm/gnu-fm-changes for Libre.fm specific changes, but almost all work is templates. https://github.com/libre-fm/libre-fm-templates and cron jobs outside of the scope of the repo.

This project is marked read-only.
